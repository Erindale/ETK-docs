Inputs
=======

|VECTOR_FIELD_SINGLE| Vector
   The input vector to map.

|VECTOR_FIELD_SINGLE| Translation

|VECTOR_FIELD_SINGLE| Rotation

|VECTOR_FIELD_SINGLE| Scale

|BOOLEAN_FIELD_SINGLE| Invert

Outputs
========

|VECTOR_FIELD_SINGLE| Vector
   The resulting output vector.
