*******
 Vertex
*******

.. figure:: /images/nodes-vertex.png
   :align: right

   The ETK_Vertex node.

The **Vertex** group creates a single vertex at the given coordinates.


Inputs
=======

|VECTOR| Vector
   Vector coordinate for the vertex.


Outputs
========

|GEOMETRY| Geometry
   The generated geometry.
